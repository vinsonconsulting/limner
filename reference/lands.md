### Special Note
Spirits from the Satchel manifest with aesthetics from their source realm, but rendered translucent/ethereal, floating in void.

### Procedural Generation Rules

The Bottomless Satchel is intentionally chaotic. Use these rules for visual generation:

**Background Collage:**
- The void contains floating fragments from OTHER Lands' environments
- Random chunks of oak branches (Seelie), stone walls (Ironroot), thatch roofs (Fenward), etc.
- Fragments drift, rotate slowly, fade in/out at edges
- Deep purple/black void between fragments
- No fragment should dominate — equal visual weight across sources

**Spirit Appearance:**
- Spirits take forms inspired by their source model's original Land
- A local Llama model appears as a translucent Smallfolk
- A quantized Claude appears as a ghostly Elf
- Apply 50% transparency, ethereal glow outline, slight particle effect
- If source is unknown/custom, use abstract elemental form (fire, water, air, earth, void)

**Accent Color Selection:**
- Pick 1-2 accent colors randomly from the full Land palette pool
- Colors should shift subtly over time (animation) or per-generation (static)
- Avoid clashing — if random selection clashes, re-roll

**Prompt Modifiers for Satchel:**
```
translucent ethereal figure, ghostly glow outline,
floating in void, surrounded by drifting fragments,
fragments from multiple realms, deep purple void background,
unstable shifting atmosphere, particle effects
```
